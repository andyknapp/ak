<p>
  As a Senior UX Engineer at <a href="https://www.thinkcompany.com/" target="_blank" rel="nofollow">Think Company</a>, most of the past year has been agentic AI development. I've been the primary contributor on a set of full-stack proof-of-concept apps and internal tools, working across agentic workflows, RAG architectures, MCP server development and integration, multi-provider LLM APIs, and AI-augmented design-to-code pipelines. I've been a leader in Think Company's internal AI literacy and adoption efforts - turning research into starter kits, templates, and internal tools the wider team can use. A few highlights:
</p>

<ul>
  <li>
    <strong>Project Initiation app</strong>A full-stack, multi-tenant agentic application that automates our client onboarding workflow. Architected a milestone-driven workflow engine that dispatches agentic and deterministic capabilities across the initiation sequence. Persona-tailored capabilities include document generation and semantic search. Features RAG document ingestion via ChromaDB, SSE-streamed LLM responses, Supabase Auth with RLS, and multi-provider LLM support.
  </li>

  <li>
    <strong>Figma Sync to Storybook</strong>A CLI tool that automates syncing of Figma design tokens, component structure, and CSS into Storybook codebases. It integrates with the Figma Desktop MCP server and includes a webhook listener that triggers syncs on "Ready for Dev" status changes.
  </li>
  <li>
    <strong>Storybook AI design system</strong>Production-ready React + TypeScript template that wraps figma-sync, an automated token pipeline, Figma Code Connect, and embedded Claude Code skills that guide AI-assisted component implementation. Designed to be cloned and configured for any client design system.
  </li>
  <li>
    <strong>Agentic full-stack starter kit</strong>Express 5 + React 19 + ChromaDB with multi-provider LLM support, MCP integrations, SSE streaming, and multi-format document ingestion. It's the foundation under several of the projects above.
  </li>
  <li>
    <strong>AI-powered WordPress theme generator</strong>Produces complete block-theme installations (theme.json, FSE templates, SQLite database) from a structured form, using a tool-calling agent to orchestrate file generation and theme activation.
  </li>
  <li>
    <strong>Statement Analyzer</strong>An internal research and strategy tool for analyzing and synthesizing purpose statements. Semantic vector search, multi-step agentic workflows, multi-format document ingestion, and OCR for scanned PDFs.
  </li>
</ul>

<p>
  I also do broader UX engineering work - building accessible, performant websites, apps, and design systems for clients, including production Storybook component libraries in JavaScript and TypeScript. The Figma-to-code AI tooling above grew directly out of that design-systems work.
</p>
